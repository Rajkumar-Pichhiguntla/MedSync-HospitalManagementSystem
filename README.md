# Hospital Management Microservices

## Architecture
This project is built using microservices architecture.

## Services
- Eureka Service (Service Discovery)
- API Gateway
- Patient Service
- Appointment Service
- Bed Service
- Staff Service
- Notification Service

## Tech Stack
- Java
- Spring Boot
- Spring Cloud (Eureka, Gateway, Feign)
- MySQL

## How to Run
1. Start Eureka Service
2. Start all microservices
3. Start API Gateway
4. Access APIs via Gateway (http://localhost:8080)#MedSync-hospitalmanagementsystem
